# Java WebService

Simple Spring Boot project demonstrating REST and SOAP student services.

- Prerequisites  
  Java 17+, Maven, (optional) Postman or curl.


- Build & run  
  mvn clean package  
  mvn spring-boot:run  
  App default URL: http://localhost:8080

-Glossary
SOAP, REST, JAXB, WSDL, endpoint, payload
